## Description

[Nextjs](https://github.com/vercel/next.js) client application for products. 
  
## Installation

```bash
$ git clone
$ npm install
```
## Update .env PORT and MONGO_URI at root level with your own credentials
```bash
PORT=YOUR_PORT
MONGO_URI=YOUR_MONGO_ATLAS_URI
```
## Start for development

```
$ npm run dev
```
